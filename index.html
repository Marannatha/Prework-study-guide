HTML

  • The head element contains information about the webpage.
  • The style element defines the CSS styles associated with a website.
  • The base element is the base URL for a webpage.
  • The meta element defines metadata like character set, description,keywords,author, and viewpoint.
  • The body element represents the visible content shown to the user.
  • HTML attributes defines additional information about an element.
  • HTML attributes are always defined in the begining element.
